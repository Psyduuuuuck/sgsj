# 3.5 宗师装备制作的建议 <a id="s246-d8877043"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

## 3.5.1 心理准备 <a id="s247-05b0e707"></a>


之所以要把心理准备放到第一个来讲，是因为宗师的制作和喂养周期实在太长了，

不肝不氪不是老玩家有老底的话，做宗师简直遥遥无期（即使只需要70%经验）。

在你家族有不小的空地的情况下，猛肝一个月左右能做出一身防具的白板。

每天肝的话30~45 天可以将一件宗师防具喂到太初（和平时间，内战可能得更久）

如果你还要做武器的话，你还需要准备圣物杖和神赐杖，这又是一笔不小的开支。

宗师里只有饰品是简单的，1 个月3 个觉醒圣物绰绰有余，但是接下来的版本饰品也可以解放了，虽然加成不算特别高，但是也要神赐才可以，因此不建议喂出圣物

0%经验的饰品。解放的神赐宗师饰品属性和现在苏醒太初属性是一样的。

其实吧，刨除运气因素，种宗师还是比买要贵不少的，宗师就和买房一样，全款最便宜，拉长时间还房贷其实是贵的，但是如果你要享受种的过程，跳经验的乐趣，

那就又不一样了。买杖其实也一样，有钱大量买低价杖其实比种狗粮还省，问题是：

你有金吗？

## 3.5.2 物质准备 <a id="s248-c76f8737"></a>


大量的空地——种地（看看做的材料表和喂的材料表你就明白了）

大量的跑商材料——跑金币

大量的领地币——制作宗师经验石头

## 3.5.3 金币准备 <a id="s249-9df05192"></a>


做喂宗师没有金币是万万不行的，没金币怎么跑黑炭龙骨、怎么买霞光树、怎么买别人的领地币。而且就算你打算自己种，种子也得好几万金买呢。特化也得不少金呢。

以下笔者为你提供两种方案并为您计算了成本，你可以按需调整。

### 偏肝方案 <a id="s250-ff1b0e7b"></a>


按一身防具四叶草玫瑰花各120 组，羊毛150 组，对应黑炭18 组，这部分成本，

种子+羊棚成本6W 金左右，200J 挂一次委托，搓50 组布料成本10W 金，挂细须委托成本36036 金。18 组黑炭按地灵货算，41 一个货要4390 包货，按一船21 个算，

需要210 船，一个货按买货26 金算，交易品成本要114146 金。

以上还是基础的花销，算上后续林林总总的，挂委托至少准备近20W 金，买交易品跑黑炭龙骨要准备至少20W 金，种地薅羊毛准备至少6W 金。

菱石不考虑打本，火木靠买霞光树（你的领地币搓狗粮还要大几百个呢），按一颗霞光树300 金出15 个火木算，要224 颗霞光树，需花67200 金买。强化菱石变动大这里就不统计了但几万金还是需要的。

亚纳碎石按定格4J 扫拍卖要60480 金，庭院粉末算3 金一个要84000 金。

按6000 一组珊瑚草药准备66000 金。

也就说在你自己疯狂种地+疯狂跑商+扫一部分材料+挂委托+有跳经验的情况下，

一身太初防具，光经验部分成本就要73W 金。

然后还有觉醒，解放和精练，这里怎么也得花100W 金。

但是这套拉长时间战线的方案，可能需要你肝1-2 年！注意是肝，不是玩！

偏肝方案需要差不多200W 本金一身精练防具。

### 偏买方案 <a id="s251-51bebf61"></a>


纯买黑炭龙骨可以大幅度减少时间，但是需要100W 金起步的花销。

找人种四叶草玫瑰花各120 组，自己种剩下的东西，大约需要30-40W 金，大批量订购不会太便宜，能多少看你讲价能力了。

委托求快可以从200 变成挂250，这样子整体花销只多5W 变成25W，但是时间上也会快不少。

亚纳碎石和烈火原木以及庭院粉末纯买花销大约50W 金。

领地币纯买方案还算便宜，最多花5-6W 金。

偏买方案精练一身防具花销大概需要300-350W 金。

## 3.5.4 关于巨匠双手杖 <a id="s252-3d71baa8"></a>


圣物杖和神赐杖，只要你要喂武器就避不开这两东西，不过好在多数情况下，神赐杖价格其实和同经验神赐狗粮的价格差不多。

喂饰品不需要巨匠杖做狗粮。

远古杖8500 一根的话1 经验成本1.16 金，8000 一根的话1 经验成本1.1 金，

7500 的话1 经验成本1.02 金。

不朽杖16000 一根的话，1 经验成本2.06 金，14000 的话1 经验成本1.8 金。

圣物杖5W 一根的话，1 经验成本为6 金，5.8W 一根的话1 经验成本为7 金，

6W 一根的话1 经验成本为7.3 金。

神赐杖8W 一根的话，1 经验成本为9.3 金，9W 一根的话1 经验成本为10.5 金。
