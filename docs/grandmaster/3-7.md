# 3.7 宗师披风 <a id="s262-24dafa9c"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

## 3.7.1 什么是宗师披风 <a id="s263-8471173b"></a>


宗师披风最早是用公会贡献换的，改版后在荣誉商店用荣誉换，是绑定的装备。

下图红框内的披风升级后就是宗师披风：

## 3.7.2 宗师披风能否交易或者转移给小号角色？ <a id="s264-0f34469e"></a>


宗师披风是绑定的，且无任何手段转移到同账号下其他角色。

## 3.7.3 宗师披风的种类 <a id="s265-fabe1cd5"></a>


宗师披风根据对应的职业属性可以分为以下5 种：

①风向披风：弓手

②骄傲披风：坦克

③庇护披风：奶妈

④闪光披风：刺客

⑤安息披风：施法法师

## 3.7.5 宗师披风经验的获取 <a id="s267-7f08e4b2"></a>


宗师披风可以用在亡灵箱子开出来的披风喂、签到送的经验石头喂、刷领地给的石头喂、原大陆石盘喂。除了亡灵箱子的披风，建议喂给石盘，将石盘喂到太初0 经验再喂给披风，这样子如果跳经验的话可以省很多的肝，经验石头喂石盘跳经验概率也更高。

## 3.7.6 宗师披风的词条 <a id="s268-3783888f"></a>


各个职业的披风都有一个固定的词条，只受披风阶段影响。各个职业披风还有一个二选一绿色词条，受品质和经验值影响。所有披风另外有4 个通用蓝色词条，受品质和经验值影响。在下图替换界面即可查看词条，以及满经验时的词条属性。
