# 2.5 玩家推进活动 <a id="s204-0cb47e06"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

## 2.5.1 什么是玩家推进活动 <a id="s205-0082b85c"></a>


需要玩家通过各种游戏内活动来满足活动开启条件的，都称之为玩家推荐活动。

玩家活动如：击杀该地图怪物，完成该地图任务，击杀玩家等。

## 2.5.2 玩家推进活动的种类 <a id="s206-1caf436a"></a>


根据触发任务类型分为以下两种：

①荣誉类（鲸鱼、烛台）

②债券类（十字星平原与伊尼斯泰尔的战争跑图）

③开启BOSS 挑战类

## 2.5.3 玩家推进活动的参与 <a id="s207-d70a133b"></a>


当鲸鱼烛台达到5 阶段后，或者有人开始组织推这些地图的战争阶段后就可以加入他们的团队。

注意：在有国人团队时，请不要加入敌对老外团队，以免被误伤。

## 2.5.4 玩家推进活动的收益 <a id="s208-15c3d683"></a>


所有类型的推荐活动都可以获得荣誉。

在玩家有接取公会每日任务的情况下，鲸鱼可以在打完扎卡鲁后获得2 个豆奶，

烛台可以获得古代修复卷轴碎片X3。

跑图任务在达到100 活动分数，并且和平到来时仍在对应地图的话。可以通过出现的蓝颜商会的NPC 来接取债券任务，任务接取就可以提交。可以获得10 个德翡纳之星和3 个债券。
