# 2.4 周常活动（weekly） <a id="s193-163e99b8"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

## 2.4.1 什么是周常活动 <a id="s194-f9fb942d"></a>


游戏内每次经过星期（一周时间）刷新时间就可以接取任务的活动为日常活动。

## 2.4.2 周常活动的种类 <a id="s195-f3cdd757"></a>


根据提升对象，可以将周常分为以下两类：

1）古代周常

2）升星周常

## 2.4.3 周常活动的参与 <a id="s196-78561c33"></a>


每周日的日常团队会自动变成日常+周常团队，其中下午4 点的XAN 团（西大陆）

是全流程团，从1 阶段古代周常开始打，所以会非常的磨叽。周日的团队位置会非常紧张，建议提前进团且不要中途退出。

## 2.4.4 周常活动的收益 <a id="s197-c0070d28"></a>


通过完成周常任务我们可以获得以下内容：

1）各阶段古代瓶子

2）各阶段古代觉醒卷轴

3）庭院瓶子

4）娜刹什之证
