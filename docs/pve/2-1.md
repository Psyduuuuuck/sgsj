## 2.1.1 副本的种类 <a id="s131-2f2aecd1"></a>


副本根据性质可以分为以下五种：

①普通副本

②英雄副本

③跨服副本

④团队副本

⑤单人副本

其中可以单刷获得最大收益的有：

①（不推荐）

②⑤

## 2.1.2 普通副本 <a id="s132-d88d96bb"></a>


随着角色等级逐渐解锁的副本，

分别为：

王宫下水道、风刃废矿、燃烧之城武器库、哈迪尔农场、呼啸深渊、毁灭之源。

但是游戏内说这些名字一般指代他们的英雄级副本，即所谓的H 本。

8.5 更新后娜刹什也拥有了普通副本和英雄副本的区别，娜刹什伽尔是推荐击杀普通副本的，因为英雄副本单刷难度过大。

## 2.1.4 英雄副本的收益分析 <a id="s134-3f4487a5"></a>


单刷副本毕业参考时间（以法师为例，从过图结束开始计时）如下：

武器库100s

农场185s

废矿150s

下水道190s （能跳过所有屏障）

下水道280s（不能跳过所有屏障）

呼啸470s

呼啸只刷第一个BOSS 蛇需要100s

娜刹什伽尔F 130S

英雄副本每天可以免费进入3 次，之后需要用副本刷新券才能再次进入，进入次数越多，所需要的券就越多。除了武器库，其他英雄副本第四次刷取都为1 张刷新券，

武器库为3 张。

掉落分析如下：

副本路上小BOSS 必掉冬日或者夏日其中一个

路上小BOSS 必掉一个防具（武器库和农场为武器）

必掉沉甸甸的袋子

必掉碎片和时装碎石

概率掉未鉴定的饰品

鉴定后分解给饰品源晶碎片

副本关底BOSS 必掉一个武器和一个防具（武器库和农场除外）、必掉蕴含灵魂的祝福石、概率掉血石（享受掉落BUFF）、必掉冬日或者夏日其中一个。

40H 小怪中概率掉防具

40H 第一个BOSS 蛇概率掉落一件防具，箱子必定掉落一件防具。

第二个BOSS 概率掉落3 个防具（一般掉落2 个）

第三个BOSS 必定掉落一把武器并概率掉一个防具

第四个概率掉两件防具

关底BOSS 必定掉落一把武器、一件防具、古神教宝箱、战宠、概率掉落血石娜刹什伽尔的英雄本BOSS 不掉落时装经验石头碎片，其余奖励和F 本一样。

娜刹什伽尔除了关底BOSS 直接掉落装备和血石，其他BOSS 只有时装经验石头，

装备在打完BOSS 后出的箱子里。

守门老一的箱子出坐骑装备，可以搞个坐骑蹄子用用，其他的无用。

老二箱子出墟境迷宫的珍贵保管箱，并概率出卡尔米拉的回忆（耳环）。

老三箱子出一把武器和四件防具和1-2 个（武器/防具）强化菱石，概率掉落一个防具箱子（至少出一件防具）或者一个武器箱子（至少出一把武器）或者一个墟境迷宫补给箱。

关底巴尔基必定掉落一把武器和一个血石、概率额外掉落第二把武器。

娜刹什伽尔副本掉落的武器分解了直接给武器源晶精华，防具分解给防具源晶晶片，饰品分解给饰品源晶晶片。

### 沉入深渊的埃尔纳图书馆 <a id="s138-4386085e"></a>


老一老二老三如果队伍输出足够就不用担心机制。

老一的机制是BOSS 身上会出现有颜色的光球，你需要去触碰同颜色的光球来避免BOSS 的高额伤害。

老二是会出线状闪电，延迟伤害，奶妈可以提前缓回，或者走位避开也行。

老三是会出一个大书怪

老四是最容易翻车的BOSS，老四，需要一个人拿休息室NPC 旁边的杖，用来堵上BOSS 房间出现的裂缝，裂缝会越来越大，召唤出来的小怪也会越来越多。

老四最重要的机制就是旋转大炮，BOSS 会先变成球形回到地图中心，然后向仇恨最高的人方向开始开炮，每炮伤害倍增，够肉可以逆旋转方向硬吃2 炮，或者吟游加速跑过去，你要是实在搞不明白或者操作不过来，就在老四房间切成带铁壁的天赋，

看到大炮就直接开无敌。

### 永恒之岛 <a id="s139-379bee6b"></a>


输出足够爆炸的话，有没有戒指都没影响。不是毕业装备队伍最好还是带上戒指（伊尼斯、公主梦、大象戒指其中一个）。出小怪清小怪即可。

### 烟飨之庭 <a id="s140-7a2e9522"></a>


（新版本烟飨可以直接快速进入）

输出够走中间大路，不够走旁边机制小路。不管走那边都要记得打完怪后拿箱子里的钥匙开门。

想打伊修达尔的话，需要饕餮、阿莉亚、白娘子都要打过高级获得BUFF 才能开门。

饕餮、阿莉亚、白娘子不管什么难度机制其实都是一样的，不过高级机制给你的容错范围更少，伤害更高。

饕餮需要站在没有尖刺石板的地面上，BOSS 掉一定血后会把所有人弹开并减速，

需要快速离开地图边缘，不然会死。

阿莉亚在唱歌时，地面上会出现颜色方块，出现红色和绿色方块后，需要站到绿色方块上。

白娘子会喷蛇毒，不一直站在蛇毒上就不会石化。白娘子还有一个机制是出高额减伤然后出灯笼怪，需要站在灯笼怪的篮圈里面打死灯笼怪，然后走到后面的楼梯爬上去按F 交互解除一个减伤符咒，重复至清除全部符咒即可。

打伊修达尔注意不要掉下平台就好。

### ①红龙（RD） <a id="s142-6f981738"></a>


1 红龙（RD）

红龙属于人多比较好打的BOSS，只要准点去排就好。可以进入时间为：

中国时间一三五七晚上7：30 和11：00；一二四六早上8：00。

萌新进去后避免站在人堆被BOSS 技能砸死就好，或者可以带一个意志，点上反伤，帮助团队更快击落红龙。红龙死亡时人在150M 内即可复活舔尸。

如果你想加快红龙下来的速度，那么你还可以准备一个乌龟（需在成团前召唤），

### ②卡杜姆（kadum） <a id="s143-f818bd86"></a>


2 卡杜姆（kadum）

这个BOSS 难度较高，如果你们团里人都不都你机制导致人死完了，那么可能到黄金你们都打不死这个BOSS，但是知道机制后这个BOSS 难度其实也就一般。（尤其是现在版本装备好很多，难度其实以及大大下降，30 人都能很快打死）

卡杜姆的进入时间和红龙刚刚好岔开，如下：

中国时间二四六七晚上7：30 和11：00；一三五七早上8：00。

卡杜姆技能和机制如下：

卡杜姆右手强击之后藤蔓范围大概在75 米，用无敌技能或者无敌翅膀或者提前往外飞都可以避免被束缚，奶妈可以直接用光与暗自救或者救别人（本质是束缚DEBUFF，能解束缚就没事），时间不是很紧张，不是太脆不用怕。

藤蔓技能之后会出现水泡，boss 后会有树枝升起、地上会出现蓝色圈圈，蓝色圈圈就是水泡落下的位置，落下之后会形成水泡陷阱，这个水泡陷阱几乎必死，因为在水泡里面至少会掉80%的血（可用霸气解除）。

boss30%的血之后boss 会关闭祭坛，此时祭坛上的人无法下来切会有2-3 波的无右手强击的预警的藤蔓。

需要注意的是别掉进祭坛下的水里，死亡后在祭坛复活点另一端召唤坐骑下去，

滑翔翼会因为死喘无法使用。

BOSS 击飞太高可使用滑翔翼或者前跳后跳来避免掉落伤害。

## 2.1.10 战场副本（跑海必看） <a id="s147-950c90ce"></a>


战场副本指的是在副本界面里战场一栏的所有项目，包括海战、角斗场、烟飨等。

攻略里将主要介绍海战副本，内容包括如何刷海战币和如何使用海战币兑换道具

### 海战 <a id="s148-43154fe7"></a>


进入海战的方式如下：

点开后选择队伍加入，然后准备即可。

注意！！！你可以在势力和跨服说嗨战，但是请不要把队伍链接发送出去。

在进海战前，记得先在任意一个地图的拍卖师那接任务（就是进幻想岛门旁边的那个鲁鲁），其中有一个任务是要连续赢3 把，先输的队伍在第四把开始前要重新接取连续赢3 把的任务。

要求连续赢3 把的任务如下图：

按照惯例，第一把进去船皮肤是蓝色的队伍先输3 把。

然后在打完5 把后，第六把开始前，游戏会问你是否重置，如果你第一把是蓝队，也就是第四把才开始做连续赢3 把的任务。这时候你需要一个道具，如下图：

重置一轮需要3 个这种币，打征兆给。

进入海战之后，找船上去驾驶，不管输还是赢都不要在家里待着。赢的队伍驾驶船之后去找海面上的剑，用船碰剑获取输出技能，输的队伍跟着队友上去送就行。

注意！！打完一把结束时，如果死了，等复活再点结束出去，不然出去后还是死的。

### 兑换海战币道具 <a id="s149-fe15d65f"></a>


海战币的兑换地点有2 个，自由岛和光芒势力主城，其中海盗只能去自由岛兑换。

兑换台子如下红框：

自由岛的兑换台子和上面的外形一样。

主要兑换道具为以下两个：

第一个为海战船头，要330 个币，不能和箱子的亡灵收集器一起使用，而且加速幅度也不如箱子的，但是是范围BUFF，还加开炮速度，适合打海战。

第二个是隐身灯，要280 个币，对比手工灯优点在于：①轻，不会因为额外增重太多而减速②隐身结束可以立马续上，不需要像手工灯一样准备2 个应对空白期。
