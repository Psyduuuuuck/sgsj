# 2.6 神之庭院 <a id="s211-46041b64"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

## 2.6.1 什么是神之庭院 <a id="s212-6b8e2cf6"></a>


神之庭院是一个类似副本的地方，不可以唤出载具，不可以挂机。只能通过世界之脐来进入。是完成古代装备养成的重要地方。同时你如果要做某些任务，也需要你的主线剧情来到神之庭院。

## 2.6.2 怎么进入神之庭院 <a id="s213-9f0399d0"></a>


玩家可以通过悉拉玛山脉东部的世界之脐的庭院之门来进入神之庭院。

注意：如果想要换称号或者敲鼓什么的，请在庭院外进行，进去后就不能进行这些操作了。

进入方法如图所示：

走向箭头指向的红框内的门按F，即可进入庭院。

## 2.6.4 精灵之庇护的获得 <a id="s215-f8a911ef"></a>


提到刷庭院，庇护是避不开的。不过当然版本庇护获取非常容易，价格比较低廉。

庇护有3 种获得方式：

①自主制作

②拍卖行购买

③每周庭院侦查分数排名奖励

目前来说是不推荐第一种方式的，因为只要开始打庭院，就有排名参与奖，会补充一些庇护，而且庇护在拍卖行价格远低于其制作成本
