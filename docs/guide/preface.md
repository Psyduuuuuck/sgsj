# 前言 <a id="s001-c441a2ce"></a>


本攻略的制作初衷是为了给内向/不喜欢语音/不喜欢被大佬辱骂的新老玩家提供一份详尽的图文攻略。

因为上古的内容真的非常多，NA 服上还有许多的未翻译内容，老玩家即便有模糊的记忆，对于各种细节可能仍有遗漏，以致于在答复萌新的提问时不够详尽。本攻略就是为了解决这个问题而制作的。

本攻略基于上古世纪NA 服7.5 版本制作，现已更新至9.5 版本，夏时令对于的中国时间为标准制作，如遇上冬时令，请自觉将中国时间向后顺延一个小时。

由于国人势力在不同时期活跃情况不同，本攻略对于世界BOSS 内容将有所取舍。

由于作者本人体验深度和能力有限，部分观点和推荐可能带有主观影响，甚至于出现一些错漏，如果您不喜欢请轻喷。当然您也可以在QQ 群或者游戏内联系作者我。

期待您的斧正。
