## 1.2.1 跑商的种类 <a id="s017-87bcb425"></a>


跑商根据交通方式的不同，可以分为以下两种：

①跑陆商②跑海商

根据货物的不同，可以分为以下两种：

③跑普通货④跑星星货⑤跑住民货⑥跑商团货

⑦跑债券货⑧跑发酵货⑨跑黑炭⑩跑龙骨

③④⑤⑥⑦⑧在跑完8 小时后，收益的金币将会通过邮件发放。

⑨⑩在跑完8 小时后，收益的资源将会通过邮件发放。

## 1.2.3 跑商对于时间的要求 <a id="s019-521227ca"></a>


所有的货物在制作出来或者购买出来时都会有一个时间，将鼠标移动在货物上即可查看（不同货物过期时间不一样）。如图所示：

因此当最佳时间结束之后，我们跑商的收益就会受损，所以我们需要根据实际情况来判断我们能不能在过期前跑完这些货物。

如果是发酵货物，那么货物还需要一个发酵时间，在发酵时间到达之前贩卖货物

的话一样会损失收益。如图：

蓝颜商会运输品（债券货）比较特殊，没有过期时间，你什么时间做都可以，但是放在地上依然存在消失时间。

现实时间推荐：

下午可以跑海

全天候可以跑安全区

## 1.2.6 打劫与被打劫 <a id="s022-ac786398"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

### 打劫的情况： <a id="s023-7373da34"></a>


如果你或者国人发现敌对势力玩家正在进行非安全区的跑商活动，可以通过语音（KOOK）或者打字的方式提醒并邀请其他国人玩家参与打劫。

### 被打劫的情况： <a id="s024-14a8a436"></a>


如果你想要进行非安全区的跑商行为，那么笔者强烈推荐你在跑商开始前就挂在语音频道上，这样子你一出事就可以立马组团、报点、报对面职业和人数已经打劫方式。同时还能告知救援人员你的情况，这样子做足了这些准备，才能最大程度保证救援人员可以帮你抢回你的货物。

但是有时候对面人数过多，如果你的船被开出了安全区，这种情况下也是抢不回来你的货物的，这种情况就认了吧。

还有一个注意事项！！！

那就是跑商一定要锁船、锁车。如果救援玩家听到你因为没有锁船锁车而被抢劫（尤其是在安全区），那么救援玩家的救援积极性将会大大下降。

最后还要提一点：

其他玩家能来救你是情分，不是本分。不管能不能抢回来你的货物，你都应该感谢他们而非指责他们救援不到位或者不及时。

你可以通过语音或者打字来表达你的谢意。相信我，其他玩家更愿意去救嘴甜的、

人缘好的人，而不是嘴臭、人缘差的人。

## 1.2.11 发酵货的制备和运送 <a id="s037-484f78e6"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

### 制备发酵架 <a id="s038-ccdec092"></a>


跑发酵需要先做置物架，制作置物架熟练度对收益无影响，因此萌新可以通过帮大佬搓发酵架来获取经商熟练度。

如果你想要长期跑发酵，那么最好有一片地（自己的或者野外的）来种大量的果树和牛棚，不然发酵消耗材料很多，扫拍卖行材料或者拍卖行发酵架会比较亏。蜂蜜则是推荐便宜时拍卖行多扫点来用，不建议自己弄。

### 制备发酵货 <a id="s039-dcb9909b"></a>


发酵的放置方法为右键背包内的置物架，放置在地里，然后走进按F 敲出发酵背在身上。注意：这时候要加速敲的读条虽然是畜牧熟练度和畜牧晨露，但是收益是按这时候你的经商熟练度算的，所以这时候你既要穿晨露腰带也要穿晨露鞋子。当然你也可以吃熟练度卷轴敲半小时发酵，交货再吃一个卷轴，以获取最高价格。（跑商熟练度最高可达35.2W）

敲好货后摆在地上即可，8X8 地极限摆货为48 个。

发酵所需时间为23 小时，但是不同地图发酵完成后的新鲜时间不同，黄金、珊瑚、地狱、西风等地图的新鲜时间较长，都为2 天左右。

发酵货运送分为陆运和海运两种，一般优质的海运发酵地，地皮都会比较贵，前期建议白嫖公会的家族的，有钱了再考虑买发酵地皮。

### 原大陆发酵相关 <a id="s040-1b060f02"></a>


由于原大陆的发酵货率极烂（50%40%等），原大陆发酵只能在地里放到过期看能不能刷新出超级发酵，超级发酵一个几千金，但是其他过期+腐烂的发酵一个只能卖十几金，连架子钱都回不来。

而且原大陆全程非安全区，需要跑车送货才比较安全，但是在居住区太里面的话，

跑车又会被各种东西卡住导致风险变大，所以你想跑原大陆发酵的话，还需要居住区边缘的土地和一辆跑车。
