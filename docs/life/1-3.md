# 1.3 钓鱼 <a id="s055-9bf9e348"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

## 1.3.1 钓鱼的种类 <a id="s056-63e8aa92"></a>


钓鱼可以根据方式不同，分为以下几类：

①钓小鱼②钓大鱼

根据内容不同，①钓小鱼可以分为以下几类：

③普通钓小鱼④自由岛高级钓鱼⑤镜之地冰钓

根据内容不同，②钓大鱼可以分为以下几类：

⑥淡水钓大鱼⑦海水钓大鱼

## 1.3.2  钓鱼的前置准备 <a id="s057-c266063a"></a>


钓鱼前根据种类的不同我们要准备的东西也不同。

如果是要钓小鱼，则需要准备鱼竿和蠕动的蚯蚓。

如果是要出海钓大鱼，则需要你必须至少有渔船、钓竿（等级越高越好）、金色鱼钩和卡夫饭团诱饵（鱼钩和饭团都可以在渔船上的制作台制作），没有船体验极差。

然后最好你再穿一个钓鱼时装，最后你需要通过渔船上的探测器找到鱼群，在游戏画面中看到海面上有海鸥飞舞的地方即是可以钓大鱼的鱼群位置。

钓鱼时装效果为减少渔船加速技能CD 和增加钓大鱼伤害。如下：

渔船制作台可制作内容如图：

如果是要淡水钓大鱼，则需要你必须至少有钓竿（等级越高越好）、金色鱼钩和有蚯蚓味饭团（可在渔船自带商店购买），有无渔船体验差别不大，且淡水钓大鱼只推荐做成就，因为收益极低。渔船自带商店内容如图：

如果是要自由岛高级钓鱼，则需要你有23W 的钓鱼熟练度以及对应的称号，还要有至少3 级的钓竿。还需要你准备有腥味的饭团（可以在渔船制作台制作）。

不管是淡水钓大鱼还是海水钓大鱼，都需要先用金色鱼钩激活鱼竿2 小时，激活方法为右键金色鱼钩，然后左键鱼竿会弹提示：

点击确认即可。

如果是要镜之地冰钓，则需要削冰铲和12W 钓鱼熟练度。

## 1.3.3 影响钓鱼时间的因素 <a id="s058-c1dacbb3"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

### 影响钓小鱼时间的因素 <a id="s059-2b50679b"></a>


1）使用床睡觉后获得的生活相关活动施法时间减少的BUFF

2）熟练度时装自带生活相关活动施法时间减少的效果

3）玩家熟练度减少施法时间

4）对应晨露时装在连续3 次钓鱼后的减少施法时间的BUFF

5）家具提供的生活相关活动施法时间减少的BUFF

6）快速生产卷轴的BUFF

### 钓大鱼多出以下影响因素 <a id="s060-8098711b"></a>


7）钓鱼时装带来的伤害加成

8）熟练度带来的伤害加成

9）鱼竿等级带来的伤害加成

10）蚂蚁、小象BUFF 带来的移速加成

11）脚蹼带来的游泳速度加成

注意事项！！！

刚刚换上鱼竿40S 内不能对大鱼使用鱼竿技能，即无法造成伤害。如图：

大鱼逃脱过远则会超过技能使用距离，需要手动接近大鱼再次使用技能延续BUFF 时间。

如果是团队钓大鱼记得让队长更改拾取，否则会出现没法捡起自己钓的鱼的情况。

## 1.3.4 钓小鱼 <a id="s061-27442109"></a>


在背包里准备充足的蠕动的蚯蚓后，装备上鱼竿，右键第一个交互键，再左键一个离自己较近的水面即可开始自动钓小鱼。

小鱼有以下2 种分类：

1)水质分类：小鱼分为淡水鱼和海水鱼，淡水鱼就是你随便哪个地图找的小沟小河，钓上来的都是淡水鱼。

海水鱼就是当前地图紧贴着大海的水域，比如水上住宅的居住区，钓上来的鱼都是海水鱼。

2)气候分类：上古世纪的每个地图，气候都是存在差别的。不同的气候，钓上来的鱼不一样。

气候主要分成四个：温带、热带、寒带、亚热带。

【一般每个地图的气候都是单一的，但是原大陆是存在双气候或者三气候的地图，

远大路这边没有测试，所以这边只说一下东西大陆。】

如果是需要挂鱼油，最适合的是亚热带的海域。比如黎明半岛左边的海域，但又不到黄金之海的地方。如图所示：
