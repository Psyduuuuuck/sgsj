# 1.6债券 <a id="s082-eb778e5e"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

## 1.6.1 债券的种类 <a id="s083-5c19dbbb"></a>


债券根据获得方式，可以分为以下两种：

①蓝颜商会债券证书

（俗称的债券一般指的是这个）

②原大陆地区发展之证

## 1.6.2 蓝颜商会债券证书的获取 <a id="s084-f7c74355"></a>


蓝颜商会债券证书可以通过以下五种方式获取：

1）住民面板每日刷新资材任务

2）地区住民NPC 每日资材和制作特产任务

3）升级房屋自带的债券任务

4）十字星平原和伊尼斯泰尔战争跑图任务

5）周六早上的攻城、守城参与奖励

## 1.6.3 原大陆地区发展之证的获取 <a id="s085-c59f3ec4"></a>


原大陆地区发展之证可以通过以下几种方式获取：

1）原大陆前线基地面板每日刷新袋子箱子提交任务

2）原大陆住民前线基地每日活动任务（击杀该地区50 个怪）

## 1.6.4 蓝颜商会债券证书的用途 <a id="s086-2127b949"></a>


蓝颜商会债券证书的用途主要为以下几种：

1）制作蓝颜商会运输品（也就是债券货）

2）提交给拥有住民资格的地区NPC 获取地区贡献，以参与地区发展3 阶段半个

月后的分红

3）在主城大厅处的噜噜NPC 处换取巨匠装备

## 1.6.5原大陆地区发展之证的用途 <a id="s087-2533539c"></a>


原大陆地区发展之证的用途主要为以下几种：

1）在光芒海岸烟飨之庭附近的NPC 处兑换古代装备（不推荐）

2）提交给拥有原大陆住民资格的地区前线基地的进贡书记官获取地区贡献，以参与地区发展3 阶段半个月后的分红

3）兑换古代经验瓶子

4）兑换披风经验石头

## 1.6.6 住民地区的阶段与分红 <a id="s088-735eeccb"></a>


所有住民地区都有发展阶段（光明海岸除外），分为为：

发展等待阶段、发展1 阶段、发展2 阶段、发展3 阶段到达发展三阶段后可以解锁地区特有制作台或者商人同时半个月后会给有地区贡献的居民按贡献比例分红，并回归到发展等待阶段。

远大路住民地图可以通过消耗劳动力清扫尸体来获得贡献值，但是东西大陆只能通过提交债券来获得贡献值。

玩家可以向自己有住民资格地图的住民会馆或者前线基地派遣队消耗10 劳动力和一个债券获得10 点该地图贡献。如下：
