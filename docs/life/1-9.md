# 1.9 保管箱 <a id="s107-4e01b96c"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

## 1.9.1 保管箱的种类及其优缺点 <a id="s108-4ea16e34"></a>


保管箱分为以下四种：

①黄泉保管箱

②保管箱

③次元保管箱

④随身保管箱

黄泉保管箱

优点：可以放入绑定物品（任务道具除外），而且容量大。

缺点：一个房屋仅能放置一个，不可以调整权限，仅限自己使用。

保管箱

优点：成本低廉，且可以调整权限成家族或者公会或者非公开缺点：只能放入非绑定道具。

次元保管箱

优点：可以放入指定类型的物品，无论是否绑定。且其中的耀眼姿态次元保管箱可以被同账号内的其他角色打开，可以通过该保管箱转移时装内衣给小号。

缺点：不可以调整权限，仅限自己使用，而且只能放入指定类型物品。

随身保管箱

优点：方便整理归类随身的物品，而且取纳方便，节约大量背包格子和回家找保管箱的时间。

缺点：购入成本高昂，有些保管箱不仅需要1000 点券购买，还需要用点券买扩容券来扩容。每种随身保管箱只能放入指定类型物品，如果该类物品较少，则该保管箱对您来说用处较小。

## 1.9.2 各类保管箱的获取 <a id="s109-c247c79e"></a>


保管箱一共有六种获得方式，分别是：

1）制作

2）点券购买

3）拍卖行购买

4）开箱子获得

5）活动获得

6）成就获得（只有随身保管箱）
