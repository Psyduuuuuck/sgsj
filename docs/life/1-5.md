# 1.5 巡山与野种 <a id="s074-fc41a191"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

## 1.5.1 巡山是什么 <a id="s075-d8761e5f"></a>


巡山指的是在野外地图（多为山间）到处闲逛，意图寻找到野外的公共资源和玩家野种的资源的行为。

## 1.5.2 巡山的收益 <a id="s076-a6b812d1"></a>


可以获得其他玩家野种的资源。获得各种果实和原木。甚至是雷击木（概率极低）。

但是当野外的资源依旧显示主人名字时，采集这些资源会产生脚印，脚印被别的人玩家举报后会产生罪恶。当然你也可以用道具消除你的脚印。

道具可以在杂货商人处购买，消除一个脚印要5 个道具。

## 1.5.3 我需不需要巡山？ <a id="s077-fdee9bc1"></a>


如果你恰巧缺少某种果实，且知道在那篇区域刷新，或者知道那有野种的就可以去巡山，否则一般情况下不建议巡山，收益不是很大。当然你除了劳动力一无所有的话，巡山还是很适合你的。

## 1.5.4 野种是什么 <a id="s078-f83b676b"></a>


野种指的是将种子或者种子捆在野外（非菜地）进行种植的行为。

野种主要推荐果树，尤其是生长时间长，占地面积大，容易死亡的果树。

## 1.5.5 野种的成本与收益 <a id="s079-e3613019"></a>


使用少许劳动力将占地面积巨大的东西种植在野外，节约了土地占有成本。

不过当今版本野种存在数量上限，一个账号为100 个单位。

## 1.5.6野种的注意事项 <a id="s080-589d00e4"></a>


野种最好种植主线活动区域视野刷新范围以外，这样子可以最大程度的避免被偷东西和避免被砍伐树木。
