## 1.1.5 影响种地时间的因素 <a id="s009-a895904c"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

### 影响作物成熟时间的因素 <a id="s010-60fb8186"></a>


当玩家使用直接种植和打捆种植时，作物的成熟时间会受到种子适应气候与种植当地的气候影响。

如果气候合适，那么作物成熟后将不会变成打蔫的XXX。

如果气候不合适，那么作物成熟后将会变成打蔫的XXX。

将鼠标指针移动到打蔫的XXX 上时可以看到作物距离消失的时间，如果这个时间后仍未收割该作物的话，该作物会直接消失。

当玩家长时间未收割作物的时候，作物会变成变质的作物，这时候需要用有机肥和劳动力来救活农作物。

### 影响收割作物时间的因素 <a id="s011-61fb69a3"></a>


1）使用床睡觉后获得的生活相关活动施法时间减少的BUFF

2）熟练度时装自带的生活相关活动施法时间减少的效果

3）玩家熟练度减少施法时间

4）对应晨露时装在连续收割3 次后的减少施法时间的BUFF

5）家具提供的生活相关活动施法时间减少的BUFF

6）快速生产卷轴的BUFF

## 1.1.6 各种类种植方式的优缺点 <a id="s012-66cfe60b"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

### 优点 <a id="s013-2678e9ca"></a>


①和②这两种种子方式都可以获得作物和副产物（树捆仅能获得静电叶片）。

②相较于①和③可以直接复种同种作物，在大量种植时可以节约时间。

③这种方式成熟时间较快且不受气候影响，长期放置不会打蔫。单个产量大种植活动花费时间较短。

### 缺点 <a id="s014-f0a86bc8"></a>


①和②成长时间较长，会受气候影响，对收割时间有要求。单个作物（捆）产量较为低下，种植活动花费时间较长。

③在收获时无法获得任何的副产物

## 1.1.7 种子的获取 <a id="s015-7eb8d6ae"></a>


种子的获取方式分为以下两种：

1）在菜单栏的生活点商店处或者生活点商人处购买（不推荐）

2）在种子和树苗商人处购买

种子和树苗商人会出现在：

1）住宅区附近的NPC 聚落处

2）相逢之庭的二楼
