# 4.2 职业选择 <a id="s276-9c6e4f61"></a>


本节点为目录/结构标题，具体操作内容见其下级章节。

## 4.2.1 输出 <a id="s277-c67b9a5a"></a>


国人不缺远程/法术输出，如果你想更加有团队贡献一点可以选择刺客或者战士。

## 4.2.2 坦克 <a id="s278-94526719"></a>


国人现在还是很缺坦克的，但是坦克自己刷怪刷本不是很方便，如果你不喜欢社交，不喜欢语音的话，不是很推荐的。虽然开开口就会有人愿意帮你，但是你要是比较孤狼一点的话，还是不要选坦克了，对自己的体验不好。

而且坦克的投入成本比较高，如圣所盾、安骑士盾、烈焰之誓、英雄石头。都不是便宜的东西，而且坦克在分数达到高分前不管PVP 还是PVE 体验都比较差劲。笔者建议，可以古代玩输出职业，宗师玩坦克职业，这样子体验会比较好。

## 4.2.3 辅助 <a id="s279-59c8bf8b"></a>


泛指各类带死亡幻术并且点不少PVP 加点，国人现在也很缺，充足的辅助对于一场大型团战，其意义还是非常重大的。最好再准备一些打团用道具，如圣所盾、血之花、克拉肯杖之类的。

## 4.2.4 奶妈 <a id="s280-084ab955"></a>


不管是大奶、小奶、奶T，奶妈这东西当然是多多益善啦。

而且现在有欢乐，奶妈自己刷怪刷本速度也不差，体验比坦克好不少。
